A state-of-the-art, broadcast-style **Live Auction Management System** designed for high-stakes IPL bidding simulations. Featuring a premium **Deep Glassmorphism UI**, real-time synchronization, and comprehensive franchise management.

---

## ✨ Key Features

### 🔨 Live Auction Command Center
*   **Real-Time Bidding**: Synchronized live feed between Admins and Team Owners.
*   **Big Screen View**: Professional broadcast-style display for public viewing with high-fidelity player cards.
*   **Instant Decisioning**: Admins can mark players as SOLD/UNSOLD with automatic purse deduction.

### 📊 Professional Management
*   **Bulk Player Import**: Upload hundreds of players instantly via Excel or CSV with automatic ₹2 Cr price capping logic.
*   **Management & Coaching Staff**: Dedicated portal to manage Mentors, Head Coaches, and support staff with separate purse tracking.
*   **Team Financials**: Real-time tracking of **Player Purse** vs. **Management Purse** with interactive progress bars.

### 🎨 High-End Design
*   **Deep Glassmorphism**: Premium "Apple-style" backdrop blur (24px) for a sophisticated visual experience.
*   **Responsive Layout**: Fully optimized for Mobile, Tablet, and Desktop displays.
*   **Security Logs**: Advanced auditing of system interactions and bidding history.

---

## 🛠️ Technology Stack

*   **Backend**: Python Flask (Scalable & Lightweight)
*   **Database**: Google Firebase / Firestore (Real-time NoSQL)
*   **Frontend**: HTML5, Vanilla CSS3 (Glassmorphism), JavaScript (Real-time Polling)
*   **Libraries**: Pandas (Data Processing), OpenPyXL (Excel Integration)

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/PrajwalK01/IPL_Auction.git
cd IPL_Auction
```

### 2. Install Dependencies
```bash
pip install -r requirements.txt
```

### 3. Setup Firebase
*   Create a Firebase project at [Firebase Console](https://console.firebase.google.com/).
*   Download your service account JSON and place it in `db/credentials.json`.

### 4. Run the Application
```bash
python app.py
```
*Access the portal at `http://127.0.0.1:5000`*
